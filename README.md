## Cara Menjalankan Program

1. Clone repository
2. Masuk ke folder project
3. Jalankan perintah:

   python main.py

## Struktur Modul

- storage.py → Penyimpanan data JSON
- validator.py → Validasi input
- create.py → Menambah tugas
- delete_update.py → Edit & hapus tugas
- timeline_ui.py → Tampilan tabel tugas
- main.py → Integrasi semua modul
